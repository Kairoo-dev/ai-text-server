# SMS AI Assistant

This project is developed and operated by Kyle Herzog.

## Overview

The SMS AI Assistant is a conversational system that allows users to interact with an AI assistant via text message. Users can send a message to a phone number and receive responses powered by an AI system.

## How It Works

- Users initiate contact by sending an SMS message
- The system processes the message and generates a response
- Replies are sent back in a one-to-one, conversational manner

This system does not send marketing or bulk messages. All interactions are user-initiated.
